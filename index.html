Latest Project

Question-Answering Chatbot

For this project, I worked in a group with a more seasoned software developer, as well as another junior developer. We used natural language processing techniques, along with nltk, the stanford named entity recognizer, and SQL to create a chatbot. The user communicates with the chatbot through the team messaging app slack. In order to do this, the code for the bot runs locally, on one of our laptops, and communicates with the slack thread using the rtmbot bot engine, which uses the slack API. The bot parses questions from user input, makes a query to the appropriate SQL database, then answer the user's question. My focuses in the project were writing the database selection function, the lyrics searching function, and the messaging output formatting.

Other Recent Projects

Sentiment Analysis on Tweets
(March 2017 to May 2017)

This was a partnered project, developed entirely in python. We created an opinion classifier that would evaluate a tweet as having a positive, negative, or neutral opinion about its subject. We achieved a peak accuracy of around 64%, compared to the all-time highest accuracy for this project of 70%. In order to do this, we tested eight classification models, each after having optimized its parameters, using pandas, a data processing toolkit, sklearn, a machine learning toolkit, and nltk, a natural language processing toolkit. The database was a set of about 10,000 tweets, each classified as having either positive, negative, or neutral opinion about the subject.

Neural Network
(July 2016)

This was an automated system for recognizing handwritten images of numbers. My implementation of this model acheived around 93% accuracy. It is a two layered feed forward network of completely connected perceptrons. The dataset can be found here.

Decision Tree
(June 2016)

This was an implementation of a standard structure. This model acheived a prediction accuracy of about 74% on test data. The dataset was available free online, from bigML. The dataset is available to download here.

AI for playing moncala
(May 2016)

This artificially intelligent program played the board game moncala. The AI used a depth first search implementation of the minimax algorithm, with A-B pruning. In several runs against the both the standard, and the advanced opponant AI, my AI won.
